# Banking System Simulation (Java OOP Project)

## Overview
This project is a console-based Banking System simulation developed in Java to demonstrate core Object-Oriented Programming (OOP) principles such as inheritance, method overriding, encapsulation, and polymorphism.

The system models different types of bank accounts and supports various banking operations including withdrawals, deposits, fixed deposits, and overdraft handling.

---

## Features

### Account (Base Class)
- Stores account holder details
- Handles basic deposit and withdrawal logic
- Maintains minimum balance requirement

### Savings Account
- Transaction limit per withdrawal
- Fixed Deposit (FD) creation with interest calculation
- Enforces minimum tenure for FD

### Current Account
- Overdraft facility
- Penalty calculation on overdraft usage

---

## OOP Concepts Demonstrated

- **Inheritance** (Savings and Current extend Account)
- **Method Overriding** (Custom withdrawal logic)
- **Encapsulation** (Balance management within class)
- **Polymorphism**
- **Constructor usage**
- **Access modifiers (protected, default, static)**
